# 说明
> 项目使用自己理解的MVVM模式、将程序结构通过目录的形式分为Modles、Views、ViewModels的目录进行分类。
> 
> 主要功能在viewmodes中编写view相关的功能，通过属性、命令绑定的方式显示交互。
> 
> 以上仅是个人理解，根据自己的想法、参考别的webUI界面编写的一个AI界面小案例，后面有时间会增加更多的功能。
